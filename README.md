\# 🎮 Reflex Game



A simple reaction speed game built with C# and Windows Forms.

Players must quickly react to randomly generated paths before time runs out.



\## Features



\- ⏱️ Real-time timer

\- 🎯 Random path generation

\- 🧠 Memory and reaction challenge

\- 📊 Score calculation

\- 🔄 Restart game

\- 🖥️ Windows Forms GUI
